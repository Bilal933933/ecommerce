# Admin Files Feature

**Description:** إدارة الملفات والصور في لوحة التحكم مع رفع عبر Cloudinary.

## Key Components & Functions

| Element | Description |
|---------|-------------|
| `FileUploader` | مكون رفع الملفات مع سحب وإفلات |
| `useFileUpload` | Mutation: رفع ملف إلى Cloudinary |

## Dependencies & Context
- `@workspace/api-client` → `filesClient`
- Backend: `modules/files/`
- Skills: `admin-skill.md`
