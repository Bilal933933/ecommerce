# Files Feature

**Description:** رفع الملفات (الصور الشخصية، المستندات) عبر Cloudinary مع إدارة الجلسات.

## Key Components & Functions

| Element | Description |
|---------|-------------|
| `FileUploader` | مكون رفع الملفات مع معاينة |
| `useFileUpload` | Mutation: رفع ملف إلى Cloudinary |

## Dependencies & Context
- `@workspace/api-client` → `filesClient`
- Backend: `modules/files/` (Cloudinary integration)
- Skills: `frontend-pattern-skill.md`
